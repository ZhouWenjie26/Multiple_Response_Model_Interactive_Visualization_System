## üìä Overview

An interactive visualization tool for the **Multiple Response Model with Inter-option Local Dependencies (MRM-LD)**, an IRT framework for Multiple Response (MR) items that directly models Response Combination (RC) probabilities while capturing inter-option dependencies. More Information: https://doi.org/10.1017/psy.2025.10073

**‚ú® [Launch the app in your browser - no installation needed!](https://zhouwenjie26.github.io/Multiple_Response_Model_Interactive_Visualization_System/)**

## üéØ Features

- **Interactive Parameter Exploration**: Adjust item parameters (discrimination, difficulty) and local dependence parameters in real-time
- **Multiple Item Types**: Support for MTF (Multiple True-False), CMS (Conventional Multiple-Select), and Select-N items
- **Dynamic Visualizations**:
  - Response Combination Characteristic Curves (RCCC)
  - Item Information Functions
  - Design Matrix (Z) display
- **Conditional & Marginal Probabilities**: Toggle between conditional and marginal probability calculations
